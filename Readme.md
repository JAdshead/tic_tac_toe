# Tic Tac Toe Game
Play Tic Tac Toe in the command line.

## Install
Fork or Clone repo

## Using

`.bin/tic_tac_toe COMMAND [OPTIONS]`

Commands:
        Player 1      Player 2
  - pvp (Player   vs  Player)
  - pvc (Player   vs  Computer)
  - cvp (Computer vs  Player)
  - cvc (Computer vs  Computer)

Options:
  - h, --help                     Help
  - n, --names Player1,Player2    Set player names
